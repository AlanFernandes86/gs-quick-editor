1. A navegação deve ser feita pelos botões do sistema, o botão voltar não deve ser utilizado.
  2. A planilha deve possuir uma linha com os títulos da colunas, diferentes entre si.
  3. Os dados devem começar pela coluna "A".
  4. Os títulos das colunas devem conter somente letras, números, espaços, " . " e acentuação.
  5. Ao utilizar planilhas existentes você deve indicar a linha que possui os títulos das colunas. O sistema se referenciará a partir dessa linha.
  6. Nesta versão, ainda não é possível usar planilhas com mais de uma página, sempre será carregada a primeira página.
  7. A coluna "photos" não deve ser alterada ou removida, somente pode ser ocultada.
